# Fast-Image-Editor
## Info
A simple tool to quickly edit images:  
- Add a solid color background  
- Apply one of 10 built-in filters (Blur, Emboss, Sharpen, etc.)  
- Save the result as PNG

Built with Python & Pillow

## How to use?
1. Place your image in the same folder as the script.
2. Run the script and follow the prompts.
3. Enter a color (or RGB) and choose a filter.
4. Get your edited image as a PNG.
